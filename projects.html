Project Title: "The Apartments"- A Spring MVC and Hibernate Application to find your Dream Apartment.
Project Description:

• Developed a Web application used to advertise about rental apartments which increased sales for the real estate agents due to online advertising thereby increased their profits.

• The Front End of the web application was developed using JSPs with HTML5 for maintaining the structure of the web pages along with CSS3 and Bootstrap for styling purposes and JavaScript, jQuery for handling user interactions.

• The entire application was developed using Spring MVC framework and Hibernate for persistence purpose.

• MySQL Workbench was used as a GUI tool to interact with MySQL database to store user data.
Enviornment:

• HTML5, C33, Bootstrap, JavaScript, Spring MVC, Hibernate, MySQL, Maven, Tomcat 8

Project Title: "Budgety": A JavaScript Application to calculate monthly budget
Project Description:

• Developed a JavaScript application which calculates monthly budget based on the list of income and expenses provided to the application.

• The Module Design pattern was used to develop the application where there were 3 modules namely, Budget Controller Module, UI Controller Module and App Controller Module.

• Budget Controller module was responsible for handling the calculation of budget and maintained the data structure of the Income and Expense objects, UI Controller module was responsible to handle the UI parts of the application whereas the App Controller module acted as a link to interact between the modules.

• The front-end of the application was developed using HTML5, CSS3 for styling purposes and JavaScript was used to handle all the user interactions and business logic.
Enviornment:

HTML5, CSS3, Bootstrap, JavaScript
